# IT_Helpdesk_SQL_Project
# IT Helpdesk Ticketing System

This project is an IT Helpdesk Ticketing System implemented in Oracle SQL.  

## Project Structure
- `sequences.sql` - Sequences for auto-increment IDs
- `tables_triggers.sql` - Users, Tickets, SLAs, Worklogs, Ticket_History tables and triggers
- `sample_data.sql` - Sample users, tickets, SLAs, and worklogs
- `triggers.sql` - Business logic triggers (auto-assign, SLA check, status logging)
- `views.sql` - Active_Tickets and SLA_Summary views
- `advanced_queries.sql` - Advanced SQL queries using CTEs, window functions, and aggregation

## Features
- Auto-incrementing IDs using sequences and triggers
- SLA monitoring and breach detection
- Ticket status logging
- Active ticket reporting
- Trend and performance analysis using advanced SQL

## Tools Used
- Oracle SQL - 21c
- SQL Developer
